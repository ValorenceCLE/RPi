Instructions:
1. Locate the key fob associated with the system and get within eyesight of the system.
2. Press and hold the reset button on the key fob. The external status light should turn red.
3. Press and hold the reset button again. The external status light should turn green.
4. The system should now be reset. Please wait about five minutes for the system to come back online.
5. If the system does not come online after 30 minutes, please contact support.
Context:
A. Performing a key fob reset is a safe way to power cycle the system. This is useful when the system is unresponsive or the system is not functioning as expected.
B. It is recommended to be within 100 feet or eyesight of the system when performing a key fob reset/bypass.
C. Performing a key fob reset will not affect the system's configuration or settings.
D. The key fob must be clicked twice to perform a reset/bypass. The first click will cut power to the system, and the second click will restore power to the system.

Instructions:
1. Open the Axis Companion app on your device and sign into your site.
2. Navigate to the 'Configuration' tab on Axis Companion.
3. Click on the 'Users' tab.
4. Select the 'Add...' button and enter the email address of the new user and select permissions level of the user.
5. The new user will receive an email with instructions on how to sign into Axis Companion.
6. Open the Axis Companion app and sign in. When signing in for the first time DO NOT CREATE A NEW SITE, rather accept the invitation to join the existing site.
Context:
A. Adding users to Axis Companion allows multiple users to access the system.
B. Users will need to complete Axis 2FA to access the system. Refer to the 'Setting up Axis 2FA' FAQ for more information.
C. If any issues arise when adding users to Axis Companion, please contact support.

Instructions:
1. The folling instructions will help you set up Axis 2FA for new site users. See the 'Adding users to Axis Companion' FAQ for more information on adding users.
2. Once a new user has been added to Axis Companion, they will receive an email with instructions on how to set up Axis 2FA.
3. Open the email from Axis and follow the instructions to create an Axis account.
4. Once the account is created, the user will be prompted to set up 2FA. Follow the instructions to set up 2FA.
5. You should now be able to login to Axis Companion. If you have any issues, please contact support.
Context:
A. If you do not have access to the email associated with the Axis account, please contact support.
B. Axis 2FA is required for all users to access the system. If you are having trouble setting up 2FA, please contact support.

Instructions:
1. To add a camera you must be connected to the system locally. Refer to the 'Connecting to the system locally' FAQ for more information.
2. Once connected to the system locally, navigate to the 'Configuration' tab on Axis Companion.
3. Click on 'Add Devices' and enter the camera's IP address and credentials.
4. Axis Companion should automatically detect the camera with an IP address of '192.168.1.3'. If the camera is not detected, please contact support.
5. You will be prompted to enter the root password for the camera. Note this is different than your normal credentials.
6. Follow the on-screen instructions to add the camera to Axis Companion. This may take a few minutes.
Context:
A. Please see 'Connecting to the system locally' FAQ for more information on connecting to the system locally.
B. If you are having trouble adding a camera to Axis Companion, please contact support imeediately.
C. The root password for the camera was provided in the initial training email sent to you. Email support if you do not have the password.

Instructions:
1. Open Axis Companion and navigate to the 'Configuration' tab.
2. You should see a list of your cameras, find the camera you want to set a preset position for and click on the three dots on the right side of the camera.
3. This will open an additional menu, click on 'Advanced Settings'. Note this may take a few seconds to load.
4. Sign in with the username/password credentials for the camera. Note these are different from your Axis Companion credentials.
5. Navigate to the 'PTZ' tab on the right hand side of the screen. Then click on 'Preset Positions'.
6. Click on the 'Create preset position'. Note that whatever the camera is pointing at when you create the preset position will become the preset.
Context:
A. If a preset position is set as the 'Home Position' the camera will automatically return here after 30 minutes.
B. There is no limit to the number of preset positions that can be set. However, there can only be one 'Home Position'.
C. Modify preset position behavior such as return to home time in the 'PTZ' tab under 'Settings'.
